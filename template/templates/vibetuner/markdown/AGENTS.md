# Core Markdown Templates - DO NOT MODIFY

**⚠️ IMPORTANT**: Scaffolding-managed files. Changes will be lost on updates.

## How to Override

**NEVER modify files in this directory!** Instead:

1. Copy template to `templates/app/markdown/`
2. Maintain the same directory structure
3. Your version overrides automatically

### Example

```bash
# Core template (if exists):
templates/vibetuner/markdown/default/template.md.jinja

# Your override (CREATE THIS):
templates/app/markdown/default/template.md.jinja
```

## Template Structure

Currently, no core markdown templates are provided by the scaffolding.

This directory is a placeholder for potential future scaffolding markdown templates.

All markdown templates should be created in `templates/app/markdown/`.
